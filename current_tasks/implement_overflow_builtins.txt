Implement __builtin_add_overflow, __builtin_sub_overflow, __builtin_mul_overflow

These GCC builtins perform arithmetic with overflow detection:
- __builtin_add_overflow(a, b, result_ptr) -> bool (1 if overflow)
- __builtin_sub_overflow(a, b, result_ptr) -> bool (1 if overflow)
- __builtin_mul_overflow(a, b, result_ptr) -> bool (1 if overflow)

Also the type-specific variants:
- __builtin_sadd_overflow, __builtin_saddl_overflow, __builtin_saddll_overflow (signed)
- __builtin_uadd_overflow, __builtin_uaddl_overflow, __builtin_uaddll_overflow (unsigned)
- Same patterns for sub and mul

Implementation approach:
1. Add BuiltinIntrinsic variants for overflow operations
2. Register all variants in BUILTIN_MAP
3. Lower to IR: perform widened arithmetic and check for overflow
4. Works across all three backends (x86, ARM, RISC-V)
